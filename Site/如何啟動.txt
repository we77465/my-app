【最有可能的原因（請依序檢查）】

(1) IIS Express 沒安裝
    打開檔案總管，看 C:\Program Files\IIS Express\ 是否存在
    若無 → 開 Visual Studio Installer 修改 VS 2022，勾「ASP.NET 與網頁開發」工作負載

(2) 本機 SQL Server 服務沒啟動或實例名稱不在腳本搜尋清單
    腳本搜尋的實例名：
      .\SQLEXPRESS, (local)\SQLEXPRESS, localhost\SQLEXPRESS
      (local), localhost, .
      .\MSSQLSERVER, (local)\MSSQLSERVER
    若你的實例名不在上面 → 改 Web.config 三條 Data Source

(3) PowerShell 執行原則或編碼問題
    我已把 .ps1 改為 UTF-8 BOM + 純 ASCII 內容，理論上沒問題
    .bat 用 ExecutionPolicy Bypass 直接呼叫 PowerShell
